Less Concepts Less Website

LESS is an abbreviated Learner Style styles, is an opensource pre-processor language for CSS. LESS is a CSS pre-processor that enables customizable, manageable and reusable style sheet for website. LESS is a dynamic style sheet language that extends the capability of CSS. LESS is also cross browser friendly.


Sass Concepts Sass Website

Sass stands for Syntactically Awesome Stylesheet. Sass is an extension to CSS. Sass is a CSS pre-processor. Sass is completely compatible with all versions of CSS. Sass reduces repetition of CSS and therefore saves time. Sass was designed by Hampton Catlin and developed by Natalie Weizenbaum in 2006. Sass is free to download and use.
